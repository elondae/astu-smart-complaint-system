# astu-smart-complaint-system
A backend-driven complaint management system designed for university environments.
This system enables structured complaint submission, tracking, and role-based management using workflow automation.
Project Overview

The ASTU Smart Complaint System provides a transparent and organized platform where:

Students can submit and track complaints
Departments can manage and resolve issues
Administrators can monitor system activity
All actions are securely stored in a database
The backend is implemented using workflow automation with structured API responses for frontend integration.
